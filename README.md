# Jarvis AI Assistant
Voice controlled AI assistant built using Python, Speech Recognition and Gemini AI.

## Features
- Wake word detection (Jarvis)
- Open/Close applications
- Play YouTube videos
- Tell current time
- AI search using Gemini
- Voice responses

## Technologies
- Python
- SpeechRecognition
- pyttsx3
- Gemini AI

## Run Project

pip install -r requirements.txt
python jarvis.py
